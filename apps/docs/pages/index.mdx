# next-sitecore-ai

AI toolkit for Sitecore JSS on Next.js.

## Getting started

Install the core package in your Next.js 14 App Router project:

```bash
npm install next-sitecore-ai
```

Copy `.env.example` to `.env` and fill in your credentials.

## Packages

| Package | Description |
| ------- | ----------- |
| `next-sitecore-ai` | Core library with hooks and utilities |
| `@next-sitecore-ai/mcp-server` | MCP server for AI tooling integration |
